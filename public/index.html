Event Planner API

Dit project bevat een REST API voor het beheren van events en locations.

Algemene parameters

  • limit (verplicht) — aantal records
  • offset (verplicht) — startpositie
  • sort (optioneel) — veld om op te sorteren
  • order (optioneel) — asc of desc

Events endpoints

  • GET /api/events?limit=&offset=&sort=&order=
  • GET /api/events/:id
  • GET /api/events/search?q=zoekterm
  • POST /api/events
  • PUT /api/events/:id
  • DELETE /api/events/:id
  • GET /api/events/upcoming
  • GET /api/events/past

Locations endpoints

  • GET /api/locations?limit=&offset=&sort=&order=
  • GET /api/locations/:id
  • GET /api/locations/search?city=Brussels
  • POST /api/locations
  • PUT /api/locations/:id
  • DELETE /api/locations/:id

Alle endpoints gebruiken JSON voor request en response bodies.
